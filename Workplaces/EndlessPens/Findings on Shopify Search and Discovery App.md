# 3. How does Search and Discovery select complementary products?
Is there a way to set up conditions? For example, if customer has pen in cart, the app should suggest ink and/or paper. If customer has ink in cart, the app should suggest pen and/or paper. Etc.

```
Search and Discovery's complementary products feature mainly work by setting them up manually per product, and they're done together with Product Recommendations. The app lets us choose recommended AND complementary products with that functionality, with up to 10 products for the list of complementary products per listing.

Good news about this even if it's done manually, is that we can bulk-edit the complementary products since the app generates a "Complementary Products" product list metafield when we initially assign a single listing their complementary product(s).
```
